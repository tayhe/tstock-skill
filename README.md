# tstock-skill

OpenClaw skills for stock market analysis.

## Skills

- `tstock-data-source` - 配置数据源，获取股票基础数据
- `tstock-fundamental_analyzer` - 基本面分析（财务指标、估值）
- `tstock-technical_analyzer` - 技术面分析（均线、MACD 等）
- `tstock-risk_analyzer` - 风险评估
- `tstock-portfolio` - 投资组合管理（关注列表、策略规划）
- `tstock-workflow` - 工作流编排（股票分析自动化）

## Installation

```bash
openclaw skills add https://github.com/tayhe/tstock-skill
```
